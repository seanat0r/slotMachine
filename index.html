The Fortune Slot Machine

How does it Work?

Click on The button to start the spin. Each spin cost a minimum of 1 Coin. The more coin you set, more coins can you win. If you hit 0 Coin you lose. You can anytime reset with the refresh button underneath the spin button.
Beware that refreshing or any other activity that restart or refreshing this browser tap will delete your Coins and you need to start again.

Shortcut for keyboard

  • Enter = start a spin OR next
  • 0-9 = set your Coin
  • R = reset
